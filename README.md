# api-server
## Install Guide
> Under your python env (>3.7)
Ensure you have requirement module
```python
pip install -r requirement.txt
```
Then
```python
cd navigator
```
Run the server
```python
python __main__.py
```

## Swagger UI
**http://0.0.0.0:8080/api/v1/ui/**
